                    (HUNDREDS)                999 < 1,000         (THOUSANDS)

            compare each place value, highest to lowest
                if greater number is greater

        A. digit at the ones place (9) is greater than digit at the ones place (0)

                >    ?

==============================

1 TEN           T     =             10 = 10 ones

1 THOUSAND      Th    =          1,000 = 10 hundreds      = 100 tens

1 ARAB          Ar    = 1,00,00,00,000 = 10 ten crores    = 100 CRORES        = 1000 ten lakhs

milli shows           1,000  times smaller    => x / 1,000            = 0.00x
centi shows             100  times smaller    => x / 100              = 0.0x
deci  shows              10  times smaller    => x / 10               = 0.x
                                                x                    = x
deca  shows              10  times larger     => x * 10               = x0
hecto shows             100  times larger     => x * 100              = x00
kilo  shows           1,000  times larger     => x * 1,000            = x,000
mega  shows        1000,000  times larger     => x * 1,000,000        = x,000,000
giga  shows   1,000,000,000  times larger     => x * 1,000,000,000    = x,000,000,000

if x = metre
    1 millimetre    = 0.001 metres         =>      1000  millimetres    = 1 metre
    1 centimetre    = 0.01 metres          =>      100   centimetres    = 1 metre
    1 kilometre     = 1000 metres          =>      0.001 kilometre      = 1 metre

if x = gram
    1 milligram     = 0.001 grams          =>      1000  milligrams    = 1 gram
    1 centigram     = 0.01 grams           =>      100   centigrams    = 1 gram
    1 kilogram      = 1000 grams           =>      0.001 kilogram      = 1 gram
