Class VI, Mathematics, Chapter 6 
Collection of numbers...
    – 4, – 3, – 2, – 1, 0, 1, 2, 3, 4, ...
    is called integers

successor/predecessor
    one MORE than   given number   gives a successor
    one LESS than   given number   gives predecessor

            3                           4           5
            3 is predecessor of 4
                                                    5 is successor of 4

Integers
    -ve                         0       +ve
    move left becomes smaller           move right becomes bigger
  <--------------------------------------------------------------->
    .. -5, -4 ..                0       .. +4, +5 ..
       -5<-4,       -4<0            0<4      4<5

To add a positive integer we move towards the right on a number line
To add a negative integer we move towards the left on a number line



                    Going up is +ve

    Going left is -ve               Going right is +ve

                    Going down is -ve



Examples:

    ---------------------------------------------
        left                    right
    ---------------------------------------------
        8 steps to the left of zero = -8
        -ve                     +ve
        decrease                increase
        spendings               earnings
        loss                    profit
        cold temperature        hot temperature
        west                    east

    ---------------------------------------------
        down                    up
    ---------------------------------------------
        below sea level         above sea level
        south                   north




Part 1

    2 +  3  = 2+3 =  3  --- go right of 0, again go right of 0 => +
                            
   -2 +  3  = 2-3 = -1  --- go LEFT of 0, go right of 0 MORE   => +
    2 -  3  = 2-3 = -1  --- go right of 0, go LEFT of 0 MORE   => -
                            
   -2 -  3  = 2+3 =  5  --- go LEFT of 0, again go LEFT of 0   => -




       +2 =  2      ---------    +number     is same as  number
       -2 = -2

brackets ()    means   "of"      means   times / multiplication

    +(-2) = -2
    -(+2) = -2
    -( 2) = -2
    -(-2) =  2      ---------    - times -   is  +




Part 2 - With brackets (), brackets means times/ multiplication

    2 + +3  = 2+3 =  3  ---------    + times +   is  +
    2 +(+3) = 2+3 =  3

    2 + -3  = 2-3 = -1  ---------    + times -   is  -
    2 +(-3) = 2-3 = -1
    2 - +3  = 2-3 = -1  ---------    - times +   is  -
    2 -(+3) = 2-3 = -1

    2 - -3  = 2+3 =  5  ---------    - times -   is  +
    2 -(-3) = 2+3 =  5




Another way of thinking

additive inverse of +6  is  -6       ( same as    -(+6)   )
additive inverse of -6  is  +6       ( same as    -(-6)   )


To subtract an integer from another integer
    it is enough to
        add the additive inverse of the integer that is being subtracted, to the other integer

          5 - (-7)
        = 5 + (additive inverse of -7)
        = 5 + 7
        = 12 ( go right 2 times )

          -5 - (+7)
        = -5 + (additive inverse of +7)
        = -5 + -7
        = -12 ( go left 2 times )

Subtract a negative integer we get a greater integer    7 - (-2) = 9,   100 - (-100)= 200
