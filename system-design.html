  • I was referring to ways to manage your data.
    Some time you write data on a distributed database, but the metadata of it can be written in a SQL DB.
    Sometimes is not enough to say that you will use a distributed system, you might additionally need to somehow efficiently manage the metadata in order to get the requirements in place.
    This seems obvious or trivial, but many people completely ignore this.
